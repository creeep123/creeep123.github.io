# 事件循环 Event Loop

任务分为两种，宏任务 (macrotask)和微任务 (microtask)

Event Loop 中，每一次循环称为 tick，每一次 tick的任务如下

  1. 执行第一个宏任务，一般是 scripts 也就是主程序，然后将主程序内的异步任务分配到两种任务队列中：
    1. 宏任务(macrotask)
    2. 微任务(microtask)
  2. 执行 1个 宏任务
  3. 执行微任务，直到微任务队列清空
  4. 回到 2 ，直到宏任务队列清空
